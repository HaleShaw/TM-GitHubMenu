# TM-GitHubMenu

Tampermonkey-Add GitHub Menu

为GitHub头添加更多的菜单项，让你能够快速抵达你想要的页面。

**注意：**

如果你想在Github企业版上使用它，你需要将你的Github企业版URL添加到脚本上方的match列表中。
